拍卖规则

第一章 总则

第一条，本规则根据《中华民族共和国拍卖法》及其他有关法律，参照国际通行惯例制定。

第二条，参加本次拍卖活动的当事人必须仔细阅读并遵守本规则，对自己在拍卖活动中的一切行为负责，并承担法律后果。

第三条，特别声明：我方对所有拍品的真伪及品质以及拍卖品所存在的瑕疵，不承担任何担保责任。

第四条，最终解释权归我方所有。

 

第二章 委托人（卖家）的规定委托手续

第一条，委托人须持有有效身份证件方可入场。

第二条，委托人委托我方拍卖其物品时，拍卖品须同时交付我方保管，未经同意，不得擅自取走。

//  第三条，委托人的佣金比例

    1、若委托人与我方之前有约定的，按约定收取

    2、若委托人与我方未作约定，按  0< x <=30   5%    收取

                                    30< x <=100  4.5%

                                   100< x <=1000  3.5% 

                                  1000< x   10%（奢侈税）

//

第四条，拍品成交后，佣金直接从成交款中扣除。

第五条，委托人可以申请保留价。保留价由我方与委托人通过协商决定，双份均不得对此擅自修改。如果委托人的拍品成交价（落槌价）低于保留价，委托人有权利决定该拍品因低于保留价而流拍，或该拍品以落槌价成交。若该拍品低于保留价成交，委托方须以保留价支付佣金。若无要求，我方默认不设保留价。

//若委托人申请保留价，则须额外支付 2.5%的佣金//

   第六条，委托人委托拍品时，需签署委托拍卖协议。

 

第三章 竞买人（买家）的规定拍买手续

第一条，竞买人须持有有效身份证件方可入场。

第二条，竞买人须持有足够资产或资产证明。若无足够资金支付，拍到的拍品流拍。

//  第三条，竞买人的佣金比例 按      0< x <=30   5%    收取

                                30< x <=100  4.5%

                               100< x <=1000  3.5% 

                              1000< x   10%（奢侈税）

//

第四条，竞买人在拍到拍品后，将在会后签署拍卖成交确认书，并当场获得拍品。

 

 

第四章，拍卖人（本学院）

第一条，我方采用英格兰式拍卖，也称“增价拍卖”或“低估价拍卖”。是指在拍卖过程中，拍卖人宣布拍卖标的的起叫价及最低增幅，竞买人以起叫价为起点，由低至高竞相应价，最后以最高竞价者以三次报价无人应价后，响槌成交。

第二条，起拍价由我方与委托人协商决定。若无协商，将默认为电商价50~70%（以淘宝亚马逊等知名电商）为起拍价。

第三条，竞价阶梯（最低增幅）默认为起拍价的10%~20%，由拍卖师根据现场气氛调整。

第四条，竞买人若要加价一次，则举一次牌。若加价幅度超过一次最低增幅，举牌同时喊出自己的报价。

 

PS：拍卖小事例：

第一则，A拍卖一本书，设有保留价120元。拍买过程中，B以落槌价为60元拍得了此书。此时，A觉得这本书卖60元亏了，所以不想卖了，他可以宣布此书因低于保留价而流拍，这样一来，B就很不满意没买的心仪的书。与此同时，A也可以忍痛割爱将书以60元卖给B，如此，B就心满意足地支付给了A 60元书费，另付2.7元（60 x 4.5%）给拍卖行。拍卖行也很高兴，A要以保留价120元支付给拍卖行4.2元（120 x 3.5%），比2.7元多了1.5元。

 

第二则，一本书起拍价50元，每次加价不得低于5元。C和D 都想买，C先举了一次牌（不说话），此时这本书拍卖价达到55元，D也举了一次牌，书的拍卖价达到60元。C一想，要在气势上压倒D，所以他举了一次牌，叫价100元，这个价超出了D的心理价位，就放弃了。C就心满意足地购得此书。
